About

The Universal Unit Converter is a web application that allows users to convert values between different measurement units quickly and accurately.


Features

  • Length Conversion
  • Weight Conversion
  • Temperature Conversion
  • Speed Conversion
  • Time Conversion
  • Area Conversion
  • Volume Conversion
  • Swap Units
  • Responsive Design

Technologies Used

  • Python
  • Flask
  • HTML5
  • CSS3
  • JavaScript
